## 📖 项目简介 | Project Overview

OpenClaw学习应用是一个**闯关式学习平台**，采用游戏化设计，帮助用户轻松掌握OpenClaw智能体系统的核心机制。无论是技术小白还是开发者，都能通过碎片化学习，一步步成为OpenClaw专家！

OpenClaw Learning App is a **gamified learning platform** designed to help users easily master the core mechanisms of the OpenClaw AI agent system. Whether you're a tech newbie or a developer, you can become an OpenClaw expert through micro-learning!

## ✨ 核心特性 | Key Features

- 🎮 **闯关式学习** - 像玩游戏一样学习，解锁新关卡
- 📱 **移动端友好** - 完美适配H5页面，随时随地学习
- ⭐ **碎片化学** - 每个课程短小精悍，充分利用碎片时间
- 🌍 **双语支持** - 中英文无缝切换
- 🎯 **互动测验** - 学后即测，巩固学习成果
- 📊 **进度跟踪** - XP经验值、连续学习天数、完成度统计

## 🚀 快速开始 | Quick Start

### 开发模式 | Development

```bash
# 安装依赖 | Install dependencies
npm install

# 启动开发服务器 | Start dev server
npm run dev

# 访问 | Visit http://localhost:5173/
```

### 生产构建 | Production

```bash
# 构建生产版本 | Build for production
npm run build

# 预览生产构建 | Preview production build
npm run preview
```
